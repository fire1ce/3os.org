# Text Highlighting

| __Markdown Syntax__                      | __Result__                          |
| ---------------------------------------- | ----------------------------------- |
| ```__bold__```                           | __bold__                            |
| ```_italic_```                           | _italic_                            |
| ```==highlight==```                      | ==highlight==                       |
| ```~~strike through~~```                 | ~~strike through~~                  |
| ```^^underline^^```                      | ^^underline^^                       |
| ``` `Inline Code` ```                    | `Inline Code`                       |
| ```==_you_ __can__ ^^combine^^`too`==``` | ==_you_ __can__ ^^combine^^ `too`== |
