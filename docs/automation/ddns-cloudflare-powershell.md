# DDNS Cloudflare PowerShell Script

Changing public IP addresses can make self-hosted services difficult to reach. This PowerShell script keeps a Cloudflare DNS record synchronized with a Windows host's current public IP address.

Source code can be found at [DDNS Cloudflare PowerShell GitHub Repository][ddns-cloudflare-powershell-github-url]{target=\_blank}.
